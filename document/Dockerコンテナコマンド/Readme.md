# Dockerコンテナの実行
- `docker container run イメージ名[:タグ名]`
	- example/echo:latestイメージをコンテナとして実行するコマンド。これだとずっとフォアグラウンド（ターミナル上で）動き続けてしまう。コンテナの実行を終了したい場合はCtrl+cでOK。
	- **-dオプション**
		- バックグラウンドでコンテナを実行させることが可能。
		- 実行するとコンテナIDが表示される。
	- **-pオプション**
		- ポートフォワーディング。外からのリクエストなどを受け入れるポート番号を指定できる。
- `docker container ls`
	- 実行中のコンテナの一覧表示。
